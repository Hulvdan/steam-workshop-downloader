# steam-workshop-downloader

## Требования для запуска

- `python 3.8+`
- `poetry`

## Запуск для использования программы, а не для разработки

```bash
poetry install --no-dev
poetry run main.py
```

## Разработка

### Запуск

```bash
poetry install
poetry shell
python main.py
```

### Тестирование

```bash
pytest
```
